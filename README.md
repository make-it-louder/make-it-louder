# C102 게임에 미치다(게미)
## 프로젝트 소개
- 프로젝트 명 : 게임에 미치다(게미)

- 프로젝트 설명
  - 섬세한 컨트롤
  - 음성인식을 이용한 기믹 등 여러가지 기믹
  - 단순한 조작을 이용한 지형지물 극복

- 프로젝트 기간 : 2023.10.10 ~ 2023.11.24

- 참조 리소스
  - Maple Story(인내의 숲)
  - [점프킹(Jump King)](https://store.steampowered.com/app/1061090/Jump_King/)
